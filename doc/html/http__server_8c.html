Genera una función que atiende solicitudes HTTP y responde con el contenido estático especificado, utilizando http_server_static_handler().

Parameters
nameNombre de la función a generar.
typeTipo MIME del contenido (ej. "text/html").

endPuntero al final del contenido en memoria.
